# fc-hud



Export do progressbar
```lua
exports['fc-hud']:startProgbar('tekst', czas)
```

Export do notify
```lua
 exports['fc-hud']:addNotification('tekst')


TriggerClientEvent("fc-hud:notifications:add", source, "Trwałość przedmiotu jest wyczerpana")
 ```
